# Make Project
The purpose of this script is to create a generic directory structure for python projects.

Relevant file: makeproject.py

```
Directory Structure
.
└── project_name/
    ├── deprecated/
    ├── docs/
    │   └── dev_log.md
    ├── src/
    │   └── test.py
    ├── .gitignore
    └── readme.md
```
